App Banco (Versão 1.0)
Este é um aplicativo de banco simples desenvolvido em Python com a biblioteca Tkinter para a interface gráfica e Excel como banco de dados. Por favor, note que esta é a versão 1.0 e uma versão mais avançada está em desenvolvimento.

Funcionalidades
Cadastro de Clientes: Permite adicionar novos clientes ao banco de dados.
Consulta de Saldo: Permite aos clientes verificar seus saldos.
Depósito e Saque: Funcionalidades básicas para depósito e saque de dinheiro nas contas dos clientes.

Requisitos
Python 3.x
Biblioteca Tkinter
Biblioteca openpyxl (para trabalhar com arquivos Excel)
Como Executar
Certifique-se de ter o Python instalado. Você pode baixá-lo em python.org.

Instale a biblioteca openpyxl usando o seguinte comando:
pip install openpyxl

Clone este repositório:
git clone https://github.com/Samuel-Gaudencio/app_banco.git

Navegue até o diretório do projeto:
cd app_banco

Execute o aplicativo:
python main.py
